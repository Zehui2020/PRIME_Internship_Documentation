Notes

Operators

  1. === : compare if value is same + if the same data type.
  2. !== : compare if value is NOT the same + if NOT the same data type.

Superglobal Vairables

  1. $_POST: Similar to get, but doesn't show result in url
  2. $_GET: To send variable value to php
  3. $_REQUEST: Array of data from $_POST, $_GET, $_COOKIE. Not recommended to use unless no choice / want all these datas. If know you want specific superglobal vairable, just use it.
  4. $_GLOBALS: Get vairable in global scope.
  5. $_SERVER: Holds info abt headers, paths, script locations, etc.

    refer to w3schools for all elements that can go into $_SERVER

Error Handling

  1. empty() : Check if vairable is empty (doesn't exist / value is false)
  2. is_numeric(): Check if vairable is a number.
  3. filter_input(type, varname, filter): Filter varname to check for special characters, is float, etc.
  4. htmlspecialchars(): Check for special html charas like <color=red></color>. Only really need to be done when outputting data to the web.

Arrays

  1. unset($array[index]): Removes value at index (doesn't shift next indes down!)

  2. array_splice(array, startIndex, indexIncrement): Removes value of index startIndex + indexIncrement and shifts it down.

    Can also do array_splice(array, startIndex, 0, newValue) to add newValue at position startIndex. Existing value at startIndex will be shifted forward 1 index to make space. newValue can also be an array!

  3. Array can function as dictionary (also called associative array) as well. Can assign keys to each value:

    $task =[
    "laundry" => "Daniel"
    "trash" => "Frida"
]
echo $tasks["laundry"]: "Daniel"

  4. count($array): Is the same as .Length in unity.

  5. sort($array): Sort incrementaly. Does it for both alphabetical and numerical.

  6. array_push(array, value): Adds value to end of array. Only works for indexed arrays!

  7. array_pop: Pops last index of array.

  8. $array[]: Adds value to end of array. Works for both indexed & associative arrays!

    // Indexed Array
$array[] = "newValue";
// Associative Array
$array["newKey"] = "newValue";

  9. Can also have array in array!

    // Indexed Array
food = [
array("banana", "cherry"),
"watermelon",
"apple"
]
echo food[0][0]: "Banana"

// Associative Array
food = [
fruits => array("watermelon", "apple"),
meats => array("chicken", "pork")
]
echo food["meats"][0]: "meats"

// Associative Array in Associative Array
food = [
fruits => array("watermelon", "apple", "ambiguous" => "tomato"),
meats => array("chicken", "pork")
]
echo food["fruits"]["ambiguous"]: "tomato"

Debugging

echo: Prints onto web page. Limited functionality. E.g. cannot print out arrays. print_r: Prints onto web page, but can print out arrays & other data types.

Built In Functions

String

  1. strlen: String length. Returns length of string (chars).
  2. str_word_count: Words in string.
  3. strpos(string, stringToFind): Returns index of stringToFind within string. If stringToFind has more than 1 char, returns the index of first char.
  4. str_replace(stringToFind, stringToReplace, fullString): Replaces stringToFind with stringToReplace in fullString.
  5. trim remove white spaces.
  6. strtolower: Make str to lower case.
  7. strtoupper: Make str to upper case.
  8. substr(string, startIndex, incrementIndex): Returns string starting from startIndex and increment by incrementIndex, Inclusive of currentIndex. incrementIndex can be negative to start from end of string.
  9. explode(dividerString, string): Returns an array with dividerString as the seperator.

Math

  1. abs: Mathf.Abs() in Unity.
  2. round: Mathf.Round() round to nearest whole number.
  3. pow: Mathf.Pow() in Unity.
  4. sqrt: Mathf.Sqrt() in Unity.
  5. rand: Random.Range() in Unity.

Arrays

  1. is_array: Check if something is an array.
  2. array_reverse: Reverse sequence of array.
  3. array_merge(array1, array2): Merges array1 with array2.

string $name = "Jane Doe"
function GetName()
{
    return $name;
}
// Use this instead (tTy to avoid this unless you have a reason to. Pass in as parameter instead.)
string $name = "Jane Doe"
function GetName()
{
    global $name;
    return $name;
}

Loops

// Indexed Array
$fruits = ["Apple", "Orange", "Cherry"];
foreach ($fruits as $fruit)
{
    echo $fruit;
}

// Associative Array
$fruits = [
    "Apple" => "Red", 
    "Orange" => "Orange",
    "Cherry" => "Pink"];

// This will give value (color)
foreach ($fruits as $fruit)
{
    echo $fruit;
}
// This will give key (fruit)
foreach ($fruits as $fruit => $color)
{
    echo $fruit;
    echo $color;
}

Database (SQL)

Vairables

  1. INT: Integer from -2137483648 - 2137483648. (4 byte data)

  2. BIGINT : 8 byte data

  3. INT(11): Can add () at the back to define width of integer. 11 is default.

  4. FLOAT: Floating point number.

  5. DOUBLE: Bigger FLOAT, more bytes.

  6. VARCHAR(10): Store 10 chars as string. Cut off parts that exceed char count. Can be used for username, password, etc. (usually max 255 chars)

  7. TEXT: Store many chars. Usually for comments, blog posts, etc.

  8. DATE 2023-05-04: Create date. Need to mind formatting.

  9. DATETIME 2023-05-04 11:30:00: Create date + time. Need to mind formatting.

  10. SIGNED / UNSIGNED: Allow / Disallow for negative numbers. E.g. (INT UNSIGNED changes the range from -2137483648 - 2137483648 to 0 - 4294967295)

  11. AUTO_INCREMENT: Increments a column by 1. It's also UNSIGNED.

  12. DEFAULT: Assign default value to a column.

  13. CURRNT_TIME: Return time of current server time.

  14. PRIMARY KEY: Ensures a certain column is unique, e.g, every user's ID will be unique if:

    id INT(11) PRIMARY KEY
//or
PRIMARY KEY (id)

  15. FOREIGN KEY ... REFERENCES: A column that references a column in another table.

    FOREIGN KEY (users_id) REFERENCES users (id)

  16. ON DELETE: Triggered when referenced table gets deleted.

  17. ON DELETE CASCADE: Triggered when referenced table gets deleted. Goes to referenced field in another table and delete everything referencing said field.

  18. ON DELETE SET NULL: Triggered when referenced table gets deleted. Set foreign key to null.

PHP & SQL Integration

  1. include_once / include: include_once will do the same as include, but will check if file alr been included earlier, if yes, throw warning / include will try to find the file, and throw warning if can't find file.

  2. require_once / require: Do the same as include_once / include, but will throw error instead.

  3. INNER JOIN / LEFT JOIN / RIGHT JOIN: Join data from 1 table with another.

  4. exit: Terminate the current script. Preferablly used in scripts without any connections.

  5. die: Terminate the current script. Preferablly used in scripts with connections.

  6. Query: To prevent SQL injection, don't just add what the user typed straight into a SQL statement.

  7. ? / :: Use these placeholders inside a query string and replace them later.

  8. Basic sequence when sending SQL in PHP:

    1. Create $sql statement
    2. Prepare the $sql statement using $link to database
    3. Bind any params
    4. Execute the statement
    5. Get result using get_result()
    6. Do something with the result!
